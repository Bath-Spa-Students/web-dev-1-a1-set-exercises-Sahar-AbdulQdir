Lists

You can also make a list in html and there are three types of it

  1. Un -ordered list:
    in this you will only be able to choose shapes to number your list, here is how :
    Code:
    <ul>
    <li>Technology is the application of conceptual knowledge for achieving practical goals</li>
    <li>We are all surrounded by technology from all corners</li>
    <li>It's difficult to step ahead without technology</li>
    </ul>
    Output:
    • Technology is the application of conceptual knowledge for achieving practical goals
    • We are all surrounded by technology from all corners
    • It's difficult to step ahead without technology
    ✎you can also change the circle into square, disc(⚫), and none.

  2. Ordered-list:
    in this, you will only be able to choose numbers or letters to number on your list, here is how :
    Code:
    <ol>
    <li>Technology is the application of conceptual knowledge for achieving practical goals</li>
    <li>We are all surrounded by technology from all corners</li>
    <li>It's difficult to step ahead without technology</li>
    </ol>
    Output:

    1. Technology is the application of conceptual knowledge for achieving practical goals
    2. We are all surrounded by technology from all corners
    3. It's difficult to step ahead without technology

  3. Describtion list:
    in this, there are no numbers,shapes or letters in it,here is how:
    Code:
    <dl>
    <dt>Technology:</dt>
    <dd>Technology is the application of conceptual knowledge for achieving practical goals</dd>
    </dl>
    Output:
    Technology:
    Technology is the application of conceptual knowledge for achieving practical goals

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style like:
1.class
you call the class selector in the css file by adding(.) before it's name
Then you open curly brakets {} and write your styling codes
2.id
you call the id selector in the css file by adding(#) before it's name
Then you open curly brakets {} and write your styling code
3.div
you call the div selector in the css file by writing it's name
4.the Universal Selector
The universal selector (*) selects all HTML elements on the page.

CSS(Cascading Style Sheets)

Which is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML
CSS is designed to enable adding colors, layouts, and fonts to the website and it was developed by the W3C.

Types of cascading style sheets:

Inline Styles

An inline CSS is used to apply a unique style to a single HTML element.

Embedded Styles

An external style sheet is used to define the style for all the HTML page.

External Styles

An external style sheet is used to define the style for many HTML pages.

Imported Styles

It's similiar to external styles.

How to link a HTML file with a CSS file:

use this code an place it at the head section:
<link rel="stylesheet" href="mystyle.css"></p>
